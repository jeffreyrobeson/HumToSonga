我已成功为您构建了全栈式 AI 音乐声景匹配器（AI Music Matcher）应用程序。以下是该系统的设计概念与功能亮点：
🎨 核心设计与视觉概念
现代画境控制中心：采用大面积负空间与优雅的卡片布局，将视觉捕获、心境点选、大模型配置与经典黑胶旋转唱机完美融为一体。
高画质预设风景库：提供“落日海滩”、“晨雾森林”等一组精心挑选的版权风景图作为测试备份，即使在无法开启摄像头的浏览器或 iframe 容器中，也能一键体验震撼的视觉声景匹配效果。
黑胶动态拾音器：采用渐变深色调设计的一体式音乐播放器。在音乐播放时，黑胶唱片会以真实的角速度进行 3D 旋转，伴有光泽跃动动画。
🚀 核心功能模块
1. 多模态 AI 画境与心境深度分析
首选官方 Gemini 架构：后端默认利用 @google/genai 和最新的 gemini-3.6-flash 引擎，解析用户拍摄/上传的画面色彩、意境，并结合选择的情感心境（喜悦、宁静、忧郁、激情、浪漫、怀旧）。
自动化智能生成：分析结果包含量身定制的中文意境标题、诗意般契合的心境描述（中文）、3-5 个美学标签，以及为后续曲库搜索精准生成的多维度中英文音乐检索词（如：“浪漫 纯音乐 吉他 海浪”）。
2. 双重通道云音乐检索与高拟真音频引擎
全栈防跨域代理：所有的音频搜索与播放解析全部经由后端 /api/music/* 进行服务中转与过滤，隐藏敏感 API Key 并完美规避跨域问题。
智能双重降级路由：
主路线：率先请求 cyapi.top 的 QQ 音乐渠道进行精细化检索与单曲解析。
副路线 (Fallback)：若主渠道未检索到歌曲，或者该歌曲没有 playable 播放源，系统会自动、无缝地降级到 Meting 备用网易云曲库进行二次匹配。
终极治愈方案：若两者均因网络原因解析失败，系统会自动启动优雅兜底，加载高质量治愈性大自然声景纯音乐，确保每一次点击都有完美声景播放。
LRC 滚动歌词同步渲染器：内置强大的歌词时间戳解析器（LRC 格式），在黑胶唱机播放时，高亮突出当前时间歌词，并自动流畅滚动至正中央。
3. 灵活的大模型切换面板
在主页面显眼处设计了 “AI 驱动模型参数配置” 展开式抽屉。
支持配置自定义提供商 (Provider)、兼容接口地址 (BASE_URL)、模型名称 (MODEL)、临时 API Key。
所有配置通过本地客户端 localStorage 实现临时保存与一键清除，无缝契合 OpenAI 格式，让您可以自由接入 DeepSeek、阿里通义千问或月之暗面等国产大模型。
4. 安全管理后台面板
管理员安全登录：通过安全验证路由登录（默认账号：admin，默认密码：admin123），数据持久化保存在服务端的 /admin_db.json 文件中。
管理员凭证修改：支持在后台即时验证旧凭证并修改管理员的账号和密码。
授权 Key 生成与管理：
支持自定义前缀一键快速生成高强度随机授权 Key（如 VIBE-XXXX-YYYY）。
列表实时展示 Key 的生成时间，支持一键切换 active/revoked 状态（启用/禁用）及一键永久删除，提供完整的运营授权控制。
